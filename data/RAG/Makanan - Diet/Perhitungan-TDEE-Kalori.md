# Perhitungan Kalori Harian (BMR & TDEE)

Untuk mengatur pola makan yang sesuai dengan target kebugaran Anda, langkah pertama adalah menghitung estimasi kebutuhan energi harian Anda.

## 1. Menghitung BMR (Basal Metabolic Rate)
BMR adalah jumlah kalori minimum yang dibutuhkan tubuh Anda untuk bertahan hidup saat istirahat total (menjalankan organ vital seperti jantung, paru-paru, otak).

Rumus Harris-Benedict yang Direvisi:
- **Pria**: BMR = 88.362 + (13.397 x Berat Badan dalam kg) + (4.799 x Tinggi Badan dalam cm) - (5.677 x Usia dalam tahun)
- **Wanita**: BMR = 447.593 + (9.247 x Berat Badan dalam kg) + (3.098 x Tinggi Badan dalam cm) - (4.330 x Usia dalam tahun)

## 2. Menghitung TDEE (Total Daily Energy Expenditure)
TDEE didapatkan dengan mengalikan BMR Anda dengan faktor aktivitas fisik harian:
- **Sedentary** (jarang olahraga, kerja kantoran duduk): TDEE = BMR x 1.2
- **Lightly Active** (olahraga ringan 1-3 hari/minggu): TDEE = BMR x 1.375
- **Moderately Active** (olahraga sedang 3-5 hari/minggu): TDEE = BMR x 1.55
- **Very Active** (olahraga berat 6-7 hari/minggu): TDEE = BMR x 1.725

## Contoh Kasus Perhitungan
Seorang pria berusia 25 tahun, memiliki berat badan 70 kg, tinggi badan 170 cm, dengan tingkat aktivitas olahraga ringan (Lightly Active).

1. **Hitung BMR**:
   - BMR = 88.362 + (13.397 x 70) + (4.799 x 170) - (5.677 x 25)
   - BMR = 88.362 + 937.79 + 815.83 - 141.925
   - BMR = 1700 kkal (pembulatan)

2. **Hitung TDEE**:
   - TDEE = BMR x 1.375 = 1700 x 1.375 = 2337 kkal per hari.

3. **Menentukan Target Kalori sesuai Tujuan**:
   - **Menurunkan Berat Badan (Fat Loss)**: Defisit 400 kkal -> Target Harian = 2337 - 400 = 1937 kkal.
   - **Menaikkan Massa Otot (Muscle Gain)**: Surplus 250 kkal -> Target Harian = 2337 + 250 = 2587 kkal.
